# PhizerGroup Assets

This folder contains all the static assets for the PhizerGroup website including:

- Images (logos, photos, icons)
- Videos (promotional content, testimonials)
- Documents (brochures, catalogs)

## Current Placeholder Images

The website currently uses placeholder images from Unsplash for demonstration purposes:

- Hero carousel images
- Apartment photos
- Wine collection images
- Team member photos
- Partner logos

## Adding Real Assets

To add your real assets:

1. Replace placeholder images with actual photos
2. Add your company logo
3. Include wine product images
4. Add apartment photos
5. Replace partner logos with actual logos

## Recommended Image Sizes

- Hero images: 1920x1080px
- Apartment photos: 800x600px
- Wine bottle images: 400x600px
- Team photos: 300x300px
- Partner logos: 200x100px
